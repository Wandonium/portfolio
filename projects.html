  • Background Generator

    Simple web tool for generating gradient backgrounds

    Background Generator is a web tool that helps you to generate gradient backgrounds for your websites. It does this by letting you pick two colors from the color pickers and then combining the selected colors into a gradient background. It also lets you pick from any two random colors by clicking the random button. The tool then displays this gradient as its own background and also provides the CSS code needed to use the chosen background.
